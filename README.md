# Constellation

Spatial memory game. A star pattern lights up briefly. Then the lines vanish and you must reconnect the same stars in the same order.

## Play

- Watch the highlighted stars in order
- Tap them in the same sequence after the pattern hides
- Each round adds a star

## Build

```
flutter pub get
flutter run
flutter build apk --release
```
